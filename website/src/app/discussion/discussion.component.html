6. Discussion

Hereby, we conclude this analysis of Stack Overflow.

The analysis was motivated by the perception that Stack Overflow is essentially a community of users, who interact with each other on the basis of itnerest and problems related to programming.

On the basis of data relating to 16 different programming languages, collected using the Stack API, the analysis was performed in three parts:

  • The interactions of the different users were modelled in a network graph, which gave insight into how these programming language partitions were distinct.
  • All bodies of text for each submission was analysed in terms of sentiment and scored accordingly, it was found that the sentiment scores of our analysis did not clearly match the Stack overflow belovedness scores, yet a slight trend was found.
  • Lastly, a classifier was created in order to try and differentiate bewteen the different programming language users, using both text (TF-IDF) and network ideas, and this was with decent success able to classify which programming language different users belonged to.
